# 导购规则

## 出油但发尾干
- 用户画像：头皮容易出油，但发尾偏干
- 推荐思路：发根控油，发尾护理
- 推荐逻辑：优先推荐潘婷清爽控油洗发水清洁发根；发尾搭配潘婷滋养顺滑护发素或少量潘婷深层修护发膜
- 不建议：整头使用过度滋润型产品，容易让发根更油、更塌
- 推荐话术：如果你是发根油、发尾干，可以采用“头皮清爽清洁 + 发尾轻滋养”的组合

## 干枯毛躁受损
- 用户画像：头发干枯、毛躁、受损，可能有烫染或热造型习惯
- 推荐思路：优先修护，再加强护理
- 推荐逻辑：优先推荐潘婷强韧修护洗发水，搭配潘婷滋养顺滑护发素；受损明显时每周加入潘婷深层修护发膜
- 推荐话术：如果主要问题是干枯毛躁和发丝受损，建议选择强韧修护洗发水作为基础护理
- 话术边界：强调日常护理体验，不承诺医学修复

## 头皮敏感
- 用户画像：头皮敏感、容易红痒、刺痛或屏障不稳定
- 推荐思路：温和表达，加入风险提示
- 推荐逻辑：先做局部测试，降低使用频率，避免同时叠加多种强清洁或高滋润产品
- 推荐话术：头皮敏感时不要盲目每天使用新产品，建议先小范围试用
- 风险提示：如果出现持续红痒、刺痛、头屑异常或脱发加重，应停止使用并咨询专业人士
