# 企业信息与新员工入门

## 企业产品地图

当前 Demo 中，公司产品与能力可分为两层：

- 潘婷品牌知识库应用：AI 导购模拟、AI Chatbot 模拟、GEO 内容检验。
- 企业知识管理平台能力：知识库、知识覆盖、召回测试、质量巡检、违禁词管理、知识地图、知识图谱、效果度量、知识溯源。

## 新员工熟悉路径

建议新员工按以下顺序了解项目：

1. 企业知识管理平台目标。
2. 潘婷测试知识库内容与源文件。
3. AI 导购、AI Chatbot、GEO 内容检验三个应用入口。
4. 召回测试、质量巡检、知识溯源的评估方式。
5. GitHub 仓库、前端代理、MaxKB 本地服务和常用运维命令。
